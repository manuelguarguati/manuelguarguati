## 🚀 Proyectos destacados

### 🤖 [TECHUS](https://github.com/manuelguarguati) — SaaS de gestión con IA
> Plataforma que permite a negocios gestionar sus páginas de Facebook, programar contenido y generar imágenes con IA.

- ✅ Integración con **Gemini AI** (fallback a OpenAI)
- ✅ Generación de imágenes con **Hugging Face**
- ✅ Sistema de suscripciones con **PayPal** y **Nequi**
- ✅ 4 tiers de planes, RLS con Supabase, deploy en Vercel
- 🛠️ `Next.js` · `Supabase` · `Tailwind` · `Vercel`

### 🎬 [CineClub](https://github.com/manuelguarguati) — App de gestión de club de películas
